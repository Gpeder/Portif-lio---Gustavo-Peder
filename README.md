# GP Portfolio

Este é um projeto React + TypeScript + Vite, configurado com Tailwind CSS v4 e totalmente limpo (sem arquivos ou pastas sobressalentes).

## 🚀 Tecnologias
- **React 19**
- **TypeScript**
- **Vite 8**
- **Tailwind CSS v4**


## 🛠️ Desenvolvimento
Para rodar o projeto localmente:
```bash
npm run dev
```

Para gerar a build de produção:
```bash
npm run build
```
